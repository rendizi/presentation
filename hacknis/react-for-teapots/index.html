What is React?

  • A JavaScript library for building user interfaces
  • Component-based architecture
  • Developed and maintained by Facebook
  • Efficient rendering with the Virtual DOM

Setting up a React Project

  1. Install Node.js: 
    node -v
  2. Install Create React App: 
    npx create-react-app my-app
  3. Navigate to the project folder and start the development server: 
    npm start

Props and State

Props

  • Short for "properties"
  • Used to pass data from parent to child
    •                   
function Welcome(props) {
  return 
    Hello, {props.name}!
    ;
}
  • Manages component-specific data
  • Stateful components can re-render based on state changes
    •                     
    
import React, { useState } from 'react';
                        
function Counter() {
    const [count, setCount] = useState(0);
                        
    return (
        <div>
            <p>Count: {count}</p>
            <button onClick={() => setCount(count + 1)}>Increment</button>
         </div>
    );
}

React Hooks

  • Use state and lifecycle features in functional components

Examples:

  • useState: Manage local state
  • useEffect: Side effects like data fetching 
              
import React, { useEffect, useState } from 'react';

function FetchData() {
  const [data, setData] = useState([]);

  useEffect(() => {
    fetch('https://api.example.com/data')
      .then(response => response.json())
      .then(data => setData(data));
  }, []);

  return 
{JSON.stringify(data)}
;
}
